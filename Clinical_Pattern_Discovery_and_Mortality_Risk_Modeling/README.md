# Clinical Pattern Discovery & Mortality Risk Modeling

## Project Overview
This project applies data mining and machine learning techniques to discover clinical patterns associated with patient mortality risk. Using multimodal clinical data from the Symile-MIMIC dataset, the analysis explores relationships between patient characteristics, laboratory values, imaging signals, and health outcomes.

The goal is to identify predictive patterns that may support early risk identification and clinical decision-making in healthcare settings.

---

## Dataset

The project uses the **Symile-MIMIC dataset**, a multimodal clinical dataset derived from:

- MIMIC-IV
- MIMIC-CXR-JPG (Chest X-rays)
- MIMIC-IV-ECG (Electrocardiograms)
- Blood laboratory measurements

These datasets provide structured clinical data along with physiological and imaging signals collected from ICU patients.

---

## Methodology

The analysis follows a typical data mining workflow:

1. **Data preprocessing**
   - Cleaning clinical variables
   - Handling missing values
   - Feature preparation

2. **Exploratory Data Analysis**
   - Distribution analysis
   - Feature correlation exploration
   - Identification of clinically relevant variables

3. **Pattern Discovery**
   - Identifying relationships between clinical features and mortality outcomes
   - Exploring potential predictive indicators

4. **Modeling**
   - Development of predictive models to estimate mortality risk
   - Evaluation of model performance

---

## Key Objectives
