
# Behavioral Risk Modeling

Author: Deepali Poudel  
Graduate Student – Statistics & Data Science

## Project Overview

This project explores behavioral traits such as impulsivity and sensation-seeking and their relationship to substance use patterns using statistical and data mining techniques. The analysis focuses on identifying patterns across personality traits, demographic factors, and substance abuse behaviors.

The goal of the project is to better understand how behavioral characteristics influence risk patterns and how statistical modeling can be used to uncover meaningful insights from behavioral data.

## Dataset

The dataset contains observations on individual personality traits, demographic attributes, and substance use indicators. Variables include measures of impulsivity, sensation seeking, and demographic characteristics such as age and education.

## Methodology

The analysis follows a structured data mining workflow:

1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature exploration and correlation analysis
4. Statistical modeling and pattern identification
5. Interpretation of behavioral risk factors

## Key Takeaways

The project demonstrates how behavioral and personality traits can be analyzed using statistical modeling techniques to uncover meaningful relationships between psychological characteristics and risk behaviors.

This type of analysis has applications in behavioral science, psychology, public health, and risk analytics.
